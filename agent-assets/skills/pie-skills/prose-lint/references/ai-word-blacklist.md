# AI 高频词黑名单

> 以下词汇在 AI 生成的中文小说中出现频率显著高于人类写作。
> 不是说这些词绝对不能用，而是 AI 过度依赖它们，导致文字可预测。
> 处理原则：大部分删除或替换，少数确实需要的保留。

## 一、动作/反应套话

| 禁用表达 | AI频率 | 替换策略 |
|---------|--------|---------|
| 不禁 | 极高 | 删除，直接写动作 |
| 不由自主 | 极高 | 删除，角色主动做 |
| 情不自禁 | 高 | 同上 |
| 身形一顿 | 高 | 写具体：脚步卡了一下/腿软了 |
| 脸色一变 | 极高 | 写具体变化：脸白了/脸绿了/眼睛瞪大了 |
| 身体一僵 | 高 | 写具体：后背绷直了/手攥紧了 |
| 猛地站起 | 中 | 可保留，但不要每次愤怒都"猛地站起" |
| 下意识地 | 高 | 删除，直接写动作 |
| 条件反射般 | 中 | 删除 |

## 二、感知/描写套话

| 禁用表达 | AI频率 | 替换策略 |
|---------|--------|---------|
| 映入眼帘 | 极高 | 直接写看到了什么 |
| 只见 | 极高 | 删除，直接描写 |
| 此时此刻 | 高 | 删除 |
| 霎时间/刹那间/一瞬间 | 高 | 删除，直接写发生的事 |
| 定睛一看 | 中 | 直接写看到了什么 |
| 赫然发现 | 高 | "看到了" |
| 一股X涌上心头 | 极高 | 写身体反应代替 |
| 心头一紧 | 高 | 写具体感受：胃抽了一下/喉咙发紧 |

## 三、对话标签套话

| 禁用表达 | AI频率 | 替换策略 |
|---------|--------|---------|
| 沉声道 | 极高 | "说" 或删标签 |
| 淡淡地说 | 极高 | "说" 或删标签 |
| 缓缓说道 | 极高 | "说" |
| 冷冷地说 | 高 | 写表情或动作代替语气描述 |
| 语气坚定地说 | 中 | 删"语气坚定地" |
| 若有所思地说 | 中 | 删除，用后续行为暗示 |
| 意味深长地说 | 高 | 删除 |
| 斩钉截铁地说 | 中 | "说" |

**原则**：90%的对话标签用"说"就够了。更好的做法是用行动替代标签——"他把杯子放下。'走吧。'" 比 "他淡淡地说：'走吧。'" 好得多。

## 四、比喻/修辞套话

| 禁用表达 | AI频率 | 替换策略 |
|---------|--------|---------|
| 仿佛/宛如/宛若 | 极高 | 删除或用"像"（但每段最多1次比喻） |
| 犹如 | 高 | 同上 |
| 目光如炬 | 高 | 写眼睛具体看向哪里 |
| 目光深邃 | 高 | 同上 |
| 嘴角微扬 | 极高 | "笑了" |
| 勾起一抹弧度 | 极高 | "笑了" 或删掉 |
| 嘴角勾起一抹XX的笑 | 极高 | "笑了" |
| 眉头微蹙 | 高 | "皱眉" 或写具体 |
| 剑眉星目 | 中 | 删除外貌套话 |

## 五、内心独白套话

| 禁用表达 | AI频率 | 替换策略 |
|---------|--------|---------|
| 心中暗道 | 极高 | 删除，直接写想法或改为行动 |
| 暗自思忖 | 高 | 删除 |
| 心中一凛 | 高 | 写身体反应 |
| 脑海中浮现 | 高 | 直接写想到了什么 |
| 他知道 | 中 | 谨慎使用，多数时候删除 |
| 他明白 | 中 | 同上 |

## 六、弱化副词（每千字 ≤ 3 个）

| 副词 | 处理 |
|------|------|
| 微微 | 删除，除非"微弱"是关键信息 |
| 淡淡 | 删除 |
| 缓缓 | 删除，除非速度真的很慢且这很重要 |
| 轻轻 | 大部分删除 |
| 悄然 | 删除 |
| 默默 | 大部分删除 |
| 隐隐 | 删除 |
| 微不可察 | 删除 |
| 几不可闻 | 删除 |

## 七、正式语体词汇（小说正文禁用）

| 禁用 | 替代 |
|------|------|
| 于是乎 | 删除 |
| 与此同时 | 删除，直接切场景 |
| 从而 | 删除 |
| 因而 | 删除 |
| 诚然 | 删除 |
| 综上所述 | 禁止出现在小说里 |
| 不言而喻 | 删除 |
| 一方面……另一方面 | 删除 |
| 不难看出 | 删除 |
| 由此可见 | 删除 |
| 事实上 | 大部分删除 |
| 值得注意的是 | 删除 |
| 显然 | 谨慎使用 |
| 毫无疑问 | 删除 |
| 某种程度上 | 删除 |
| 在某种意义上 | 删除 |

## 八、虚词/对冲词（每千字 ≤ 3 个）

| 词 | 说明 |
|----|------|
| 似乎 | 叙述者应该确定地说 |
| 可能 | 除非角色真的在猜测 |
| 或许 | 同上 |
| 大概 | 同上 |
| 也许 | 特别是在章末，极可能是AI结尾 |

## 九、意义膨胀词

| 禁用 | 处理 |
|------|------|
| 意义深远 | 删除，写具体影响 |
| 前所未有 | 删除 |
| 可谓 | 删除 |
| 史无前例 | 删除 |
| 划时代 | 删除 |
| 不可思议 | 谨慎使用 |

## 十、结尾套话（章末禁用）

| 禁用 | 说明 |
|------|------|
| 未来可期 | AI结尾标志 |
| 前途无量 | AI结尾标志 |
| 充满希望 | AI结尾标志 |
| 这一切，才刚刚开始 | 极高频AI结尾 |
| 他知道，接下来…… | AI结尾标志 |
| 也许，这就是…… | AI结尾标志 |
| 夜色中，XX的背影…… | AI结尾标志 |
