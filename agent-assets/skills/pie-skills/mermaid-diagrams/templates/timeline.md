# Timeline Template

## When to Use
Historical events, chronological progression, history of changes

## Basic Template
```mermaid
timeline
    title Project History
    2024
      : Project started
      : First release
    2025
      : Major update v2
      : Feature complete
    2026
      : Current version v3
```

## With Detailed Events
```mermaid
timeline
    title AI Development Timeline
    2024 Q1
      : OpenCode released
      : Multi-model support
    2024 Q2
      : Agent improvements
      : Better context handling
    2025
      : OmO v3 released
      : Agentic Coding
```

## Best Practices
- Use year, quarter, or date format
- Each event: `: Event description`
- Can have multiple events per time period
