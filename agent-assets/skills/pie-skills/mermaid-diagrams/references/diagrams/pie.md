# Pie Chart

**Keyword:** `pie`
**Best for:** Proportions, percentages, distribution

## Quick Template
```mermaid
pie title Distribution
    "Category A" : 40
    "Category B" : 35
    "Category C" : 25
```

## Label with Quotes
```mermaid
pie title "团队成员"
    "开发" : 50
    "测试" : 20
    "产品" : 15
    "其他" : 15
```

## Tips
- Quote Chinese labels
- Values sum to any number (not just 100)
- Use for simple proportions only
