# Sankey Diagram

**Keyword:** `sankey`
**Best for:** Flow magnitude, resource distribution

## Quick Template
```mermaid
sankey-beta
    title Energy Flow
    Coal, Electricity: 50
    Gas, Electricity: 30
    Nuclear, Electricity: 20
    Electricity, Home: 60
    Electricity, Industry: 40
```

## Syntax
- `Source, Target: Value`
- Shows flow from source to target

## Tips
- Good for energy, money, resource flows
- Width = magnitude
