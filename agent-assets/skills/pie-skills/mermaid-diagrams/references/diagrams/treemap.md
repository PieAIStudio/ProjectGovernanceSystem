# Treemap

**Keyword:** `treemap`
**Best for:** Hierarchical proportions, budget breakdown

## Quick Template
```mermaid
treemap
    title Budget Distribution
    "Total" 100000
      "Development" 50000
        "Frontend" 25000
        "Backend" 25000
      "Marketing" 30000
      "Operations" 20000
```

## Syntax
- `"Parent" value`
- Indent for hierarchy
- Size = value

## Tips
- Good for budget, storage breakdown
- Nested rectangles show proportion
