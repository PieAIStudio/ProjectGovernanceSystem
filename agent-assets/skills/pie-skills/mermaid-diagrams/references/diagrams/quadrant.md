# Quadrant Chart

**Keyword:** `quadrantChart`
**Best for:** Comparison, prioritization, positioning

## ⚠️ CRITICAL: Chinese Must Be Quoted
```mermaid
quadrantChart
    title "项目优先级"
    x-axis "低" --> "高"
    y-axis "低" --> "高"
    quadrant-1 "优先"
    quadrant-2 "委托"
    A: [0.3, 0.8]
    B: [0.8, 0.6]
```

## English Version
```mermaid
quadrantChart
    title "Priority Matrix"
    x-axis "Low" --> "High"
    y-axis "Low" --> "High"
    quadrant-1 "Do First"
    Project A: [0.3, 0.8]
```

## Values
- x: 0-1 (left to right)
- y: 0-1 (bottom to top)

## Tips
- ALL Chinese text in double quotes
- Values are [x, y] coordinates
- Max 4 quadrants
