# User Journey

**Keyword:** `journey`
**Best for:** User satisfaction over steps

## Quick Template
```mermaid
journey
    title User Checkout Flow
    section Purchase
      Visit page: 5: User
      Add to cart: 4: User
      Checkout: 2: User
      Payment: 1: User
```

## Syntax
- `section Name` - Groups steps
- `Step: rating :Actor`
- Rating 1-5 (5=great, 1=poor)

## Tips
- Rating shows satisfaction
- Good for UX analysis
