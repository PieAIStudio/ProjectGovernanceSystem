# Mindmap

**Keyword:** `mindmap`
**Best for:** Hierarchical ideas, brainstorming, organization

## Quick Template
```mermaid
mindmap
  root((中心主题))
    分支1
      子主题A
      子主题B
    分支2
      子主题C
```

## With Content
```mermaid
mindmap
  root((项目管理))
    规划
      需求分析
      时间线
    执行
      开发
      测试
    交付
      部署
      维护
```

## Tips
- `((text))` for root node
- Indent for hierarchy
- Chinese works well
