# Timeline

**Keyword:** `timeline`
**Best for:** Historical events, chronological progression

## Quick Template
```mermaid
timeline
    title History
    2024
      : Event 1
      : Event 2
    2025
      : Event 3
```

## With Quarter/Date
```mermaid
timeline
    title Title
    2024 Q1
      : Event
    2024-03
      : Specific date
```

## Tips
- Use year, quarter, or date
- Each event: `: Event description`
- Multiple events per period allowed
