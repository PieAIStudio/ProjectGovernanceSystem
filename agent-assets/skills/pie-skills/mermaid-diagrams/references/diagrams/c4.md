# C4 Diagram

**Keyword:** `C4Context`, `C4Container`, `C4Component`
**Best for:** Software architecture at multiple levels

## System Context
```mermaid
C4Context
    title System Context
    Person(user, "User")
    System(app, "Application")
    System_Ext(payment, "Payment Gateway")
    
    Rel(user, app, "Uses")
    Rel(app, payment, "Calls")
```

## Container Level
```mermaid
C4Container
    title Container Diagram
    Container(web, "Web App", "React", "UI")
    Container(api, "API", "Node.js", "Backend")
    ContainerDb(db, "Database", "PostgreSQL")
    
    Rel(web, api, "HTTP")
    Rel(api, db, "JDBC")
```

## Levels
- `C4Context` - System context
- `C4Container` - Containers
- `C4Component` - Components
