# ScreenWalk Advanced Challenge Panel

Use this file only for advanced runs on core, risky, or controversial screens.

Default ScreenWalk does **not** require this panel.

## Run A · First-time newcomer

- low context
- low patience
- does not fill in missing meaning for the product

Use this run to expose basic comprehension failures.

## Run B · Skeptical payer

- sensitive to charging friction
- suspicious of unclear value exchange
- notices vague premium language quickly

Use this run when pricing, upgrade prompts, or quota boundaries are in play.

## Run C · Adversarial judge

- challenges easy agreement
- looks for overconfident claims with weak evidence
- asks whether a recommendation is really grounded in the screen

Use this run to lower the chance of sycophantic or decorative feedback.

## Output discipline

After the advanced panel:

1. list what all runs agree on
2. list what only one run noticed
3. separate likely evidence from weak inference
4. call out any issue that should go back to `$goalcascade`

Do not treat multi-run AI agreement as equivalent to real-user validation.
