# 情节点：正确与错误示例

## 五种标识的实例对比

同一个故事片段，用不同标识意味着完全不同的写法：

**原始剧情**：苏明送完货在楼道里想起了石佩欣说的那句话，犹豫要不要告诉同事。

```md
❌ 错用【场景】：
【场景】苏明在楼道里想了很多，犹豫要不要说出去。
（没有可以"拍出来"的动作和事件，不该用【场景】）

✅ 应该用【内心】：
【内心】苏明站在楼道里抽了半根烟，把石佩欣那句"我知道你的工号"在脑子里翻来翻去——她是真的知道，还是在吓他？他数了一下自己能拿出什么，结论是：什么都没有。
```

```md
❌ 错用【场景】：
【场景】苏明和老赵说了好长时间，两个人谈了机器人的事。
（如果核心就是那段对话，应该用【对话】）

✅ 应该用【对话】：
【对话】老赵问他昨天那单为什么跑那么久，苏明说包太大楼道窄，老赵盯着他看了三秒，说"行"，然后转身走了——那个"行"说得很平，苏明说不出哪里不对劲。
```

```md
❌ 错用【场景】：
【场景】石佩欣把苏明推出去，关上了门。
（动作序列，极少或无对白，用【动作】更准确）

✅ 应该用【动作】：
【动作】石佩欣直接用两只手把他推出去，苏明来不及说话脚步踉跄退到门外，门砰一声关上了。
```

## 错误写法 vs 正确写法

### 错误 1：功能分析型

❌ 错误：
```md
1. 这一段展现了两个人之间的关系进一步恶化。
2. 这一章的功能是让人物更加立体。
3. 故事继续升级，人物更加复杂。
```

问题：
- 太空，不能落地
- 没有人物、动作、时间、地点
- 读完不知道"谁干了什么"

✅ 正确：
```md
1. 【场景】石佩欣从厕所出来，当着他的面把跳蛋充上电，像故意一样把东西摆在插排旁边。
2. 【场景】苏明脑子"嗡"一下，当场下单，付了七天试用押金。
3. 【概述】下完单以后，两个人谁都没服软，屋里空气僵到像马上要炸。
```

---

### 错误 2：心理医生腔

❌ 错误：
```md
1. 苏明意识到自己其实一直渴望被接纳，这种感情被压抑太久了。
2. 石佩欣内心深处害怕亲密关系，所以用强势来保护自己。
```

问题：
- 是作者在替人物解释，不是人物在发生事情
- 读者看到的是分析，不是故事

✅ 正确：
```md
1. 【场景】苏明被石佩欣嘲笑了两句，本想怼回去，愣了三秒，最后只说了句"随便"。
2. 【场景】石佩欣催他签合同，苏明说要再想想，石佩欣当场把合同推到一边，说"你这种人不值得我催"。
```

---

### 错误 3：空泛升级

❌ 错误：
```md
1. 两人的矛盾进一步激化。
2. 场面越来越失控，情绪越来越复杂。
```

✅ 正确：
```md
1. 【场景】苏明当着三个同事的面说漏了嘴，把石佩欣的工资数字说出来了。
2. 【场景】石佩欣没当场发作，只是把手机扣在桌上，等人都走完，才叫苏明留下来。
3. 【场景】苏明以为她要骂自己，结果石佩欣只问了一句："你是故意的吗？"
```

---

### 错误 4：单条塞太多

❌ 错误：
```md
1. 【场景】苏明发现了谎言，两人大吵，石佩欣承认了，两人和好，但苏明心里还是有疙瘩。
```

问题：发生了四件事，一条根本装不下。

✅ 正确（拆成4条）：
```md
1. 【场景】苏明在石佩欣手机里看到了转账记录，当场脸色就变了。
2. 【场景】两人爆发，苏明质问，石佩欣一开始否认，被追问三轮后才承认了数字。
3. 【场景】石佩欣先道歉，苏明没吭声，最后接受了，两人重新坐到了同一张桌子。
4. 【概述】表面上和好了，但苏明开始留意石佩欣的每一笔手机支出。
```

## 典型的好情节点

**同时包含动作 + 阻碍 + 转折：**
```md
【场景】两个人被逼得只能临时编一个更大的谎：说他们在做直播带货测试，还借了两个人体模特回家试拍。
```
这条里同时有：动作（编谎）、阻碍（被逼）、转折（从藏机器人升级到撒更大的谎）

**典型的【概述】用法（压缩时间）：**
```md
【概述】苏明开始偷偷盯物流，白天送快递时心不在焉，晚上回家第一件事就是看门口有没有纸箱。
```
这条有：压缩了一段时间，描述整体状态，不适合逐秒展开。

## 快速自检清单

写完情节点后，对照检查：

- [ ] 每条能回答"谁干了什么"吗？
- [ ] 有没有以"这一段……"开头的句子？（有就删）
- [ ] 有没有"心理医生腔"（渴望、疗愈、压抑）？（有就换成动作）
- [ ] 有没有空泛升级（"更加""进一步"）？（有就写具体）
- [ ] 有没有单条塞了三件以上不同的事？（有就拆）
- [ ] 用的是有序列表吗？（不是就改）
- [ ] 每条都有 `【场景】` 或 `【概述】` 标识吗？
