# 交付前自检清单

每次输出情节点之前，过一遍这个清单：

- [ ] 每条能回答"谁干了什么"吗？
- [ ] 有没有以"这一段……"开头的句子？（有就删）
- [ ] 有没有心理医生腔？（有就换成动作）
- [ ] 有没有空泛升级词？（有就写具体）
- [ ] 有没有单条塞了三件以上不同的事？（有就拆）
- [ ] 用的是有序列表吗？
- [ ] 每条都有五种标识之一（【场景】【概述】【对话】【内心】【动作】）吗？
- [ ] 标识选对了吗？对话主导的用了【场景】？纯内心的用了【场景】？（回头检查）
- [ ] 每章末尾是钩子，还是收尾句？（收尾句要改）
- [ ] 每章开头接住了上章末尾的悬念吗？
- [ ] 高强度和低强度标识有没有明显失衡？
